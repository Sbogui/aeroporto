# Sistema de Gerenciamento de Aeroporto ✈️

Projeto acadêmico desenvolvido em **Java**, simulando operações de um aeroporto.  
Inclui classes e métodos para gerenciamento de voos, passageiros e logística interna.

## Funcionalidades
- Cadastro de voos e passageiros
- Associação de passageiros a voos
- Estrutura orientada a objetos (POO)

## Tecnologias
- Java
- Programação Orientada a Objetos
